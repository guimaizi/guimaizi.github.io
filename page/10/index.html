子域名监控

域名监控概括

子域名收集这个路子真的是被玩烂了，花样百出、工具没有八百也有一千，无非是爆破、爬、调用搜索引擎之类，有资源的大厂有自己的dns库，但是这些在我眼里真的都很low。

为什么说很low，因为在职业刷src或者apt攻击者的角度，单单过一遍爆破的域名是不能满足持续性漏洞挖掘的;从职业刷src的角度，过一遍收集的子域名，已经发现了所有漏洞并已经提交后修复，或者用当前漏洞测试方法并没发现有漏洞，这样业务是安全的，但这个安全是在当下时间的，企业要发展、要解决当前问题，就会出新业务不断的pull代码更新旧问题，这就是业务的变化，通过持续性监控子域名就会发现业务的变化，最快速度的发现变化，对变化进行安全测试、漏洞挖掘。有经验的刷src的同学都知道，新业务发现漏洞概率都很高。

懒惰使人创造工具，而我就拥有这么一个工具，从14年搞的子域名爆破工具(送给过一个小姐姐，刷了好多漏，如今以身为人妻，而我还没有女朋友。)，到如今的子域名监控工具。

具体实现思路

1、通过域名爆破、搜索引擎之类方法，获得子域名后爬取子域名http响应数据保存入数据库。
2、设定时间、可以是一分钟、一小时、一天一次循环读取库内子域名，进行爬取子域名和库类http响应数据对比，对比出变化推送提醒。

从思路上是个简单的工具，但是我的集成chrome headless、subfinder，自我感觉mongodb的可视化也不错，操作便捷、使用稳定。 结果类似图上，正式版本title不会出现乱码。

功能: 通过爆破、爬取收集子域名，之后循环爬取库内数据对比发现业务的变化和新业务后及时展示、推送。 域名黑名单机制，可设置过滤.ke.qq.com、.qzone.qq.com、*.bz.qq.com之类n级子域泛解析。

金融危机流程

加息

近代股市上涨下挫、金融危机大多是从控制货币发行权的各国央行利率的变化开始，制定经济政策的政府控制着货币的供给量，在各个经济周期中给予合理的利率，保持市面适当的流动性，维护国家安定团结、健康发展。
上一轮美联储加息干死了房地产次级贷引发全球性的金融危机，在上一次金融危机发生时，美联储为了避免出现1929年的大萧条，刺激经济把利率维持在0.25%，维持7年(09-15)才开始逐渐加息，如今2018年11月利率是2.25%。
可以这样肤浅的理解什么是利率:年化利率0.25%,借一百块一年只需利息0.25块，只要投资产品年回报率高于0.25%，你就可以去找美联储借钱。
加息是为了防止低利率时出现的经济风险，进而加息开始收缩流动性达到释放化解风险的目的，也是为了度过下一次危机留有余地。

杀估值

估值就是价值，任何东西都有价值，米、面、菜、企业这些都是资产，任何资产都会有估值，估值由市场来决定，市场由供给关系决定。
在股市中可以用一些指标来估值，pe(市盈率)、pb(市净率)之类、大多数资产会用pe这个指标来估值，pe=市值÷年利润，所以可以这样理解，你投资了当前价值等于150块的资产，每年产生收益10块，十五年赚回本金，所以pe=15,你透支了十五年的未来进行投资，这十五年也可以理解为预期。
货币供给量和央行利率 放松或紧缩后，货币的流动性就会受到影响，进而影响到资产估值，市面上的资产开始资产重估，流动性放松就pe就放大，流动性收紧pe就缩小。
估值与利率之间的关系:在大米的供给不变的情况下，在低利率时一斤米是一块五毛钱，在高利率时一斤米就可能是一块钱，不是大米变了而是货币的价值变了。 因美联储和各国央行加息，市面流动性锁紧，今年各国股市指数pe就缩小了很多。

杀业绩

市面流动性收紧，投资自然就放缓，资产价格开始重估，大宗商品价格回落，人们对未来感到担忧，消费放缓，这样影响到企业利润。

违约率

流动性放缓和企业利润降低会导致高杠杆的资产破产，从而引发违约率。

失业率

紧缩的流动性和违约率引发大家更加不愿意消费，投资，没有投资就没有就业，失业率就此产生。

释放流动性

失业率后，为了安定团结，低利率时代重新开始，政府重新释放货币，刺激经济。

End

释放流动性还是不能解决就业后，人闲是非多，谁也说不准会怎样，可以参考世界上发生的经济体制改革与政治体制改革，危机驱动重大改革，成功改革开放高速发展几十年，失败的例子很多，无非是战争与犯罪率。

子域名随机收集

取随机数随机组合出(0-9 a-z .)字符串，dns穷举子域名
需要环境python3.5+ mongodb、别的自己pip install *
本来是异步的、但是requests会阻塞就线程了。
超时时间和线程数识网络环境自己设置

# coding: utf-8
'''
Created on 2018年10月15日

@author: guimaizi
'''
import dns.resolver,requests,random,queue,time,threadpool
from pymongo import MongoClient
from bs4 import BeautifulSoup
class start:
    def __init__(self,domain):
        '''
        :domian 子域名收集
        '''
        self.domain=domain
        #超时
        self.timeout=5
        #线程数
        self.thread_num=100
        self.domain_list=[]
        self.result=[]
        #每次任务标记
        self.random=self.generate_random_str(16)
    def generate_random_str(self,randomlength):
        """
        :randomlength 生成一个指定长度的随机字符串
        """
        random_str = ''
        base_str = 'abcdefghigklmnopqrstuvwxyz0123456789.'
        length = len(base_str) - 1
        for i in range(randomlength):
            random_str += base_str[random.randint(0, length)]
        return random_str

    def query(self,domain):
        '''
        dns查询存在域名
        '''
        myResolver = dns.resolver.Resolver()
        myResolver.nameservers = ['8.8.8.8', '8.8.4.4']

        try:
                myAnswers = myResolver.query(domain, "A")
                self.domain_list.append(domain)
        except Exception as e:print(e)
    def get_http(self,domain):
        '''
        http获取:
        {"domain":domain,"url":target_url,"status_code":"%s"%str(r.status_code),"size_html":len(r.text),"title":"%s"%str(self.get_title(r.text)),"mark":self.random,"time":time.strftime('%Y-%m-%d',time.localtime())}
        '''
        try:
            kv={'user_agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/69.0.3497.100 Safari/537.36'}
            target_url='http://'+domain
            r = requests.get(target_url,headers=kv,timeout=self.timeout)
            r.encoding='utf-8'
            data={"domain":domain,"url":target_url,"status_code":"%s"%str(r.status_code),"size_html":len(r.text),"title":"%s"%str(self.get_title(r.text)),"mark":self.random,"time":time.strftime('%Y-%m-%d',time.localtime())}
            print(data)
            self.result.append(data)
        except Exception as e:print(e)
    def get_title(self,html):
        '''
        返回title
        '''
        try:
            soup = BeautifulSoup(html, 'html5lib')
            title = soup.find('title')
            return title
        except:
            return 'not'
    def save_result(self,data_list):
        '''
        #保存数据
        :filename 文件路径及文件名
        :data_list list数据
        '''
        domain=self.domain.replace('.','_')
        f=open('%s.txt'%domain,'a',encoding='utf-8')
        for i in data_list:
            f.write('%s\n'%str(i))
        f.close()
    def threadpool_fun(self,fun,lists,num):
        '''
        thread多线程池

        :fun 函数
        :lists 数据
        :num 线程数
        '''
        q = queue.Queue()
        for i in lists:
            q.put(i)
        lst = [q.get() for i in range(q.qsize())]
        pool = threadpool.ThreadPool(num)
        requestss = threadpool.makeRequests(fun, lst)
        [pool.putRequest(req) for req in requestss]
        pool.wait()
        pool.dismissWorkers(num, do_join=True) 
    def run(self,list_str):
        #去重
        client = MongoClient("localhost", 27017)
        db_target_domian = client.domian
        domain=self.domain.replace('.','_')
        collection =db_target_domian[domain]
        list_domain=[]
        for i in list_str:
            i=i+'.'+self.domain
            if collection.find({"domain": "%s"%i}).count()==0:
                list_domain.append(i)
        print(list_domain)
        #运行
        self.threadpool_fun(self.query, list_domain, self.thread_num)
        time.sleep(self.timeout)
        self.threadpool_fun(self.get_http, self.domain_list, self.thread_num)
        #写入数据
        collection.insert(self.result,manipulate=True)
        time.sleep(self.timeout)
        #self.save_result(self.result)
    def random_str(self):
        list_str=[]
        for i in range(1000):
            list_str.append(self.generate_random_str(random.randint(1,10)))
        #运行
        self.run(list(set(list_str)))
if __name__=="__main__":
    while 1:
        try:
            item=start('qq.com') 
            item.random_str()
        except:continue
